# Greaseweazle v4.1 3D printed case

3D printed case model for [Greaseweazle v4.1](https://github.com/keirf/greaseweazle).

Case cut-outs designed for silkscreen visibility.

Inspired by [Greaseweazle v4.1 Slimline Case
](https://www.thingiverse.com/thing:6324390) by Tom Dalby ([Tom_DD](https://www.thingiverse.com/Tom_DD/designs)).
